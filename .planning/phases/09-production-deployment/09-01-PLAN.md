**scripts/backup.sh** — create (per D-16, D-17):
```bash
#!/usr/bin/env bash
# Daily PostgreSQL backup via docker exec.
# Usage: bash scripts/backup.sh
# Schedule via host cron: 0 2 * * * /path/to/project/scripts/backup.sh

set -euo pipefail

BACKUP_DIR="/backups"
DATE=$(date +%Y-%m-%d)
FILENAME="autodealer_${DATE}.sql.gz"
CONTAINER="autodealer-ai-assistant-postgres-1"

# Run pg_dump inside the postgres container and compress output
docker exec \
  -e PGPASSWORD="${POSTGRES_PASSWORD:-postgres}" \
  "${CONTAINER}" \
  pg_dump -U postgres autodealer \
  | gzip > "${BACKUP_DIR}/${FILENAME}"

echo "Backup written: ${BACKUP_DIR}/${FILENAME}"

# Remove backups older than 7 days
find "${BACKUP_DIR}" -name "autodealer_*.sql.gz" -mtime +7 -delete
echo "Old backups cleaned (retention: 7 days)"
```
Make the script executable. The BACKUP_DIR `/backups` is the pg_backups volume mount point inside the postgres container — but the script runs on the HOST and uses docker exec. If the operator wants to run it directly, they should set BACKUP_DIR to their local backup path and the script will work with the piped output. Add a comment clarifying this.

Actually: since the script runs on the host (per D-17), the backup file should be written to the host, not inside the container. Revise: the script pipes pg_dump output to a LOCAL path on the host:

```bash
#!/usr/bin/env bash
# PostgreSQL daily backup script. Runs on the host machine via cron or 'make backup'.
# Writes compressed backups to BACKUP_DIR on the host.
# Schedule via: crontab -e -> 0 2 * * * /full/path/to/scripts/backup.sh

set -euo pipefail

SCRIPT_DIR="$(cd "$(dirname "${BASH_SOURCE[0]}")" && pwd)"
BACKUP_DIR="${BACKUP_DIR:-${SCRIPT_DIR}/../backups}"
DATE=$(date +%Y-%m-%d)
FILENAME="autodealer_${DATE}.sql.gz"
CONTAINER="${POSTGRES_CONTAINER:-autodealer-ai-assistant-postgres-1}"

mkdir -p "${BACKUP_DIR}"

docker exec \
  -e PGPASSWORD="${POSTGRES_PASSWORD:-postgres}" \
  "${CONTAINER}" \
  pg_dump -U "${POSTGRES_USER:-postgres}" "${POSTGRES_DB:-autodealer}" \
  | gzip > "${BACKUP_DIR}/${FILENAME}"

echo "Backup written: ${BACKUP_DIR}/${FILENAME}"

find "${BACKUP_DIR}" -name "autodealer_*.sql.gz" -mtime +7 -delete
echo "Retention: removed backups older than 7 days"
```
